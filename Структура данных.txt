ТИП: список
- Название: string
- Картинка: jpg, png, gif
- Дополнительно* (например, для жгутов здесь будет ссылка на схему, по которой они сделаны. Также сюда предлагается вносить любые примечания к ТИПУ): string
- ОБЬЕКТЫ: список
	- Серийный номер: string	
	- Признак «Свободен»: boolean
	- Признак «Спецпроверка»: boolean
	- АНКЕТА: фиксированный набор полей
		- Дата изготовления: дата
		- Дата поставки не нужна?: дата
		- Инвентарный №: integer
		- Ссылка на документ поставки: string
		- Место хранения оригинала документа поставки: string
		- Ссылка на этикетку от платы: string
		- Указание где хранится оригинал этикетки?: string
		- Ссылка на необходимое ПО (например, драйверы): string
		- Указание где хранятся оригиналы ПО (на дисках)?: string
	- ИСТОРИЯ: список
		- Дата взятия: дата
		- Дата сдачи: дата
		- Кем взята: string (ссылка на профиль)
		- Куда взята: string
		- Признак верификации сдачи: boolean
		- Администратор, назначенный на верификацию: string (ссылка на профиль)
		- Комментарий (комментарий к КОНКРЕТНОМУ СЕАНСУ испоьзования): string
	- Примечание (примечание к конкретному объекту): string


«Список» означает, что подразумевается множество элементов с набором полей, представленных во вкладке.
«Фиксированный набор полей» означает, что данный раздел является как бы контейнером для набора полей, которые в него входят. Введен для удобства восприятия. 
Знаками "?" указаны поля, которые требуют обсуждения
