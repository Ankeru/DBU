Структура данных (Итерация: 2)

﻿ТИП: список
- Название: string
- Ссылка на картинку: string
- Ссылка на необходимое ПО (например, драйверы): string
- Дополнительно* (например, для жгутов здесь будет ссылка на схему, по которой они сделаны. Также сюда предлагается вносить любые примечания к ТИПУ): string
- ОБЬЕКТЫ: список
	- Серийный номер: string	
	- Признак «Свободен»: boolean
	- Признак «Спецпроверка»: boolean
	- Дата изготовления: дата
	- Дата поставки: дата
	- Ссылка на документ поставки: string
	- Место хранения оригинала документа поставки: string
	- Ссылка на этикетку от платы: string
        - Место, где хранится оригинал этикетки: string
	- ИСТОРИЯ: список
		- Дата взятия: дата
		- Дата сдачи: дата
		- Кем взята: string
		- Кем выдана (администратор, который выдал): string
		- Кем принята (выставляется при сдаче - администратор, который принял): string
		- Куда взята: string
		- Комментарий (комментарий к КОНКРЕТНОМУ СЕАНСУ использования): string
	- Примечание (примечание к конкретному объекту): string
